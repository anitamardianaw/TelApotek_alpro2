# TelApotek

## Deskripsi

TelApotek merupakan aplikasi Manajemen Stok Apotek yang dikembangkan sebagai Tugas Besar mata kuliah Algoritma dan Pemrograman 2. Aplikasi ini dibangun menggunakan Go (Golang) untuk membantu pengelolaan data obat melalui terminal.

## Fitur

- Menambahkan data obat
- Menampilkan data obat
- Mengubah data obat
- Menghapus data obat
- Mencari obat berdasarkan gejala
- Mencari obat berdasarkan ID
- Mengurutkan data berdasarkan tanggal kedaluwarsa
- Mengurutkan data berdasarkan stok
- Mengurutkan data berdasarkan ID
- Menampilkan stok terbanyak
- Menampilkan stok terkecil
- Menghitung total stok obat

## Konsep yang Diimplementasikan

- Array
- Struct
- Function
- Procedure
- Switch Case
- Sequential Search
- Binary Search
- Selection Sort
- Insertion Sort
- Rekursi
- Find Maximum
- Find Minimum
